## Cryptography Algorithms Visualizer

A Python desktop application that implements 10 classical and modern 
cryptography algorithms with an interactive GUI built using Tkinter.

---

## Features

- Encrypt & Decrypt text using 10 different algorithms
- Brute Force attack demonstration on Caesar Cipher
- Clean and simple desktop interface

---

## Supported Algorithms

### Classical Ciphers
| Algorithm     | Type |
|-----------    |------|
| Caesar Cipher | Substitution|
| Affine Cipher | Substitution|
| Vigenère Cipher| Polyalphabetic|
| Rail Fence Cipher| Transposition|
| Playfair Cipher| Substitution|
| One-Time Pad| Substitution|
| Columnar Transposition| Transposition|


### Modern Ciphers
| Algorithm | Type |
|-----------|------|
| RSA       |Asymmetric|
| DES       |Symmetric|

### Attack
| Technique | Target |
|-----------|--------|
|Brute Force|Caesar Cipher|

---

##  Tech Stack

- **Language:** Python 3
- **GUI:** Tkinter
- **Libraries:** pyDes, onetimepad, base64, math

---

##  How to Run

1. Clone the repository
   git clone https://github.com/Walaa2004/cryptography-algorithms-visualizer.git

2. Install dependencies
   pip install pyDes onetimepad

3. Run the application
   python main.py
